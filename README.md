# TRUPTI XEROX - Premium Document & Service Center

A modern, high-performance React + Vite website for TRUPTI XEROX, featuring a comprehensive suite of services including document processing, government assistance, and professional solutions.

## Project Overview

This is a static, conversion-focused website built with modern web technologies to deliver:
- **Premium Digital Experience**: Polished, intentional design with smooth animations
- **Fast Performance**: Optimized for mobile-first delivery with <2s load times
- **Service Showcase**: Clear presentation of 11+ professional services
- **User Engagement**: Interactive elements, smooth scrolling, and dynamic animations

## Key Features

### Homepage Sections
- **Hero Section**: Animated text reveal with staggered animations
- **Service Ribbon**: Infinite scrolling service carousel
- **Services Grid**: Interactive service cards with hover animations
- **Process Timeline**: Step-by-step service delivery process
- **Trust Section**: Key trust indicators and company values
- **Stats Section**: Animated counters showcasing impact
- **Testimonials**: Auto-rotating customer feedback
- **CTA Section**: Strong call-to-action with multiple contact methods

### Service Pages
- Detailed service information
- Required documents checklist
- Processing time and charges
- FAQ accordion
- Quick contact options

### UX Enhancements
- Smooth scrolling with Lenis
- Sticky navbar with glass effect
- Floating action buttons (Call, WhatsApp, Directions)
- Mobile-optimized sticky action bar
- Scroll-triggered reveal animations
- Page transition effects

## Technology Stack

- **React 18**: UI framework
- **Vite 5**: Lightning-fast build tool
- **TypeScript**: Type-safe development
- **Tailwind CSS**: Utility-first styling
- **Framer Motion**: Advanced animations
- **React Router**: Client-side routing
- **Lenis**: Smooth scrolling library

## Installation

```bash
# Install dependencies

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm npm preview
```

## Design System

### Color Palette
- **Primary**: `#FFD700` (Yellow) - Highlights and CTAs
- **Dark**: `#000000` (Black) - Headings and text
- **Light**: `#FFFFFF` (White) - Background
- **Accent**: `#FF0000` (Red) - CTA emphasis

### Typography
- **Headings**: Bold modern sans-serif (Inter)
- **Body**: Clean readable sans-serif (Inter)

## Performance Optimizations

- Lazy loading images
- Optimized animation rendering
- Clean component structure
- Minimal bundle size
- Tailwind CSS purging
- No heavy 3D libraries

## Mobile First

- Responsive grid layouts (1-4 columns)
- Large tap-friendly buttons (48px+ height)
- Sticky mobile action bar
- Proper font scaling
- Touch-optimized interactions
- Minimal animation on small screens
